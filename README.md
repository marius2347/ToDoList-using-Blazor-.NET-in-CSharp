# Blazor To-Do List ‚úÖ

A simple yet functional **To-Do List application** built with **ASP.NET Core Blazor**. This project demonstrates how to build interactive web UIs using **C#** and **.NET**, featuring real-time data binding and event handling.

## üöÄ Features

* **Task Management:** Users can add new tasks to the list dynamically.
* **Real-time Status Updates:** Mark tasks as completed using interactive checkboxes (two-way binding).
* **Dynamic Counter:** Automatically tracks and displays the number of incomplete tasks remaining.
* **Input Validation:** Prevents adding empty tasks to the list.
* **SPA Experience:** Updates the UI instantly without page reloads.

## üì¶ Getting Started

To run this application locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/marius2347/ToDoList-using-Blazor-.NET-in-CSharp.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd ToDoList-using-Blazor-.NET-in-CSharp
    ```
3.  **Run the application:**
    ```bash
    dotnet run
    ```
4.  Open your browser and navigate to the localhost URL shown in the terminal (usually `https://localhost:7000` or similar).
5.  Navigate to the `/todo` page from the side menu.
